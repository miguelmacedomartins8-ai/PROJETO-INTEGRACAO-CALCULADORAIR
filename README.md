Nosso projeto, chamado Klion, é um site que oferece uma calculadora de Imposto de Renda simples, intuitiva e totalmente funcional. O objetivo foi criar uma ferramenta acessível para ajudar o usuário a entender melhor a tributação e simular o valor do IR rapidamente. Durante o desenvolvimento, utilizamos conceitos fundamentais de programação, boas práticas de desenvolvimento web e lógica aplicada.
