Possiveis dúvidas sobre o site

Esta calculadora substitui um contador?

Não. Ela é apenas uma ferramenta didática para ajudar a entender o impacto de renda, deduções e dependentes no imposto estimado.

Os valores e faixas de imposto são oficiais?

Não. A tabela usada aqui é fictícia para fins de estudo e testes com a interface. Para cálculos oficiais, consulte um profissional ou as normas vigentes.

Meus dados ficam salvos?

Não. Esta versão de demonstração calcula tudo apenas no seu navegador, ele não armazena os resultados para um banco de dados.
